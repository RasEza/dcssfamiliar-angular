How to load a character file

To load your character file, you can press "#" in webtiles to generate the dump. It will give you an url to the character dump file. You can press "Ctrl-a" and then "Ctrl-c" in your keyboard to copy the whole content. Then you paste on the above input box and submit. (note that on mac you use the command key instead of Ctrl)

What is DCSS?

Assuming you have no idea about DCSS you can go to http://crawl.develz.org/wordpress/. It is a dungeon crawl roguelike. It is played by downloading a client or in the browser (or by using a terminal)
