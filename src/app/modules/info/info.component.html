How to load a character file

Watch a video tutorial: youtube link

Quick guide: While playing DCSS, you can press "#" to generate the dump. It will give you an url/path to the character dump file. Got to that text file and select all with "Ctrl-a" and then copy with "Ctrl-c". Then go to DCSS familiar (this website), paste the contents on the above input box, and submit. (note that on mac you use the command key instead of Ctrl)

What is DCSS?

Assuming you have no idea about DCSS you can go to http://crawl.develz.org/wordpress/. DCSS is a dungeon crawl roguelike. It is played by downloading a client or in the browser (or by using a terminal)
